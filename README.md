#Примеры структуры проекта микросервисов

#Запуск
Собрать проекты UserService.Api.csproj и OrderService.Api.csproj

Запустить сервисы:
 - UserService.Api
 - OrderService.Api

В браузере зайти на страницы:
 - http://localhost:5217/scalar - UserService.Api
 - http://localhost:5218/scalar - OrderService.Api
 
 На странице http://localhost:5217/scalar  добавить пользователя.
 На странице http://localhost:5218/scalar попробовать добавить заказ.
