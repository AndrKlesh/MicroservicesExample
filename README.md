# Примеры структуры проекта микросервисов

# Запуск
1. Собрать проекты UserService.Api.csproj и OrderService.Api.csproj

2. Запустить сервисы:
 - UserService.Api
 - OrderService.Api

3. В браузере зайти на страницы:
 - http://localhost:5217/scalar - UserService.Api
 - http://localhost:5218/scalar - OrderService.Api
 
 4. На странице http://localhost:5217/scalar  добавить пользователя.
    На странице http://localhost:5218/scalar попробовать добавить заказ.
